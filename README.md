# Euroleague Basketball Data Analysis and Prediction

This repository includes and an end-to-end methodology for building machine
learning algorithm for predicting Euroleague Basketball game outcomes.

The repository consists of the following modules (which represent logical steps
in the modelling process)

*   `data-collection`
*   `feature-extraction`
*   `descriptive-analysis`
*   `model-selection`
*   `feature-selection`
*   `model-validation`
